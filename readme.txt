Steps to open and run project1.py
1. Extract tar content in the same folder
2. Open project1.py using Microsoft Visual Studio.
3. Make sure to have the most popular python extension installed. Can use the extension tab in the left to search for it. (Ctrl+Shift+X)
4. Open the explorer tab on the left (Ctrl+Shift+E), select open folder.
5. Select the folder containing project1.py, test.txt, and train-fall2022.txt
6. Run the code using the icon the in top left (Ctrl+Alt+N)
7. Question answers should be marked and printed to console
